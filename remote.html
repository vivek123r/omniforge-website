Microphone Permissions Help

⚠️ In-App Browser Warning: QR scanner built-in web views (WeChat, generic scanners) block mic access. Tap the menu in the top-right corner and select Open in Chrome or Open in Safari.

Browsers block mic access ('getUserMedia') on local network IPs over HTTP for security. Since this portal is hosted securely under HTTPS, you simply need to allow access when prompted. If it was blocked:

For Google Chrome (Android/Desktop): Click the padlock icon 🔒 in the URL address bar, find Microphone, and toggle it to Allow.
For Apple Safari (iOS/iPhone): Open iOS Settings -> Safari -> Microphone, and ensure it is set to **Ask** or **Allow**. Then refresh the page.
